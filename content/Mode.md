A **[[Mode]]** defines the *operating posture* of a spell—how the chosen [[Hook]] is applied.  
While a Hook is the spell’s verb, Mode is the spell’s **intent**: whether Flux is being generated, altered, or actively directed.

Mode is written at the **center** of a sigil alongside the [[Hook]] and the [[Control Tier]].

## Modes (Scholar Standard)

- **Create**
  Brings an effect into existence directly from [[Flux]].
  - Use when the spell produces something that wasn’t already there (heat, force, light, sound, etc.).
  - Most “projectile” and “burst” spells are Create.
![[Mode__CREATE.png]]
- **Affect**
  Alters something that already exists.
  - Use when the spell modifies a target’s current state (warm what’s there, cool what’s there, reshape an existing flame, dampen an existing sound, harden a material, etc.).
  - Often more efficient than Create when the environment already contains what you need.
![[Mode__AFFECT 1.png]]
- **Control**
  Directs, maintains, or continuously steers an ongoing effect.
  - Use when the spell requires live guidance, sustained shaping, or active regulation over time.
  - Common with Move (telekinesis/levitation), long-lived Ward effects, or any spell with “keep it stable” behavior.
![[Mode__CONTROL.png]]
## Quick rule of thumb
- If the effect is **made from nothing** → **Create**  
- If the effect is **changing something real** → **Affect**  
- If the effect must be **kept behaving** → **Control**

## How Mode interacts with other variables
- **[[Persistence]]** tends to map naturally:
  - Create often pairs with Immediate or timed durations.
  - Affect can be immediate or timed depending on whether the change persists on its own.
  - Control commonly pairs with Sustained, Latched, or Conditional effects.

- **[[Discipline]]** and **[[Output Mode]]** stay the “how/what,” while Mode is the “posture.”
  - Example: Heat + Thermal can be used in any Mode:
    - Create: generate heat
    - Affect: warm an object that already exists
    - Control: regulate temperature over time
